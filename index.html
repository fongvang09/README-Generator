Description

The purpose of this project is to generator a README.md page that looks and feels professional.

Installation

A user of this README Generator will need Node.js and Visual Studio Code (VSC).

Usage

Within VSC, right-click on index.js or any JavaScript files and select "Open in Integrated Terminal". Type down "node index.js" in the terminal and it'll prompt the user with some questions. Answer accordingly. Upon answering the last question, a "Success!" line will appear meaning you have done this correctly. An empty "index.html", in this case, will now have lines of codes that can now be opened in the File Explorer displaying a README page that the user created. The Github link should take the user to the corresponding response that they had input (assuming if it's an active username). The LinkedIn URL should display a live link if the user entered their personal LinkedIn URL. The Table of Contents are clickable and will take you to where it is on the page. Also, a badge will appear next to the title of the page depending on which of the four badges that were selected.
